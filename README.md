# Microprocessadores-relatório

Projeto desenvolvido para a disciplina de Microprocessadores.

## Objetivo

Implementar a lógica de funcionamento de um elevador de duas cabines e quatro andares.

## Arquivos

- `codigo_arduino.cpp` - Implementação em C/C++ para Arduino.
- `codigo_mips.asm` - Tradução do algoritmo para Assembly MIPS.
- `Relatorio.pdf` - Relatório final do projeto.
